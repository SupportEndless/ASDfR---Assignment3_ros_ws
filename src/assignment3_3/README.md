Package assignment3_2
-----------------------------------------------
### Description 
This package implements the Firm real time part of assignment 3.3.
It contains the Xenomai 20-sim node using the FSM system. Requires the 
XRF2 bridge node and the relbot_adapter node to function properly. Also
contains the controller generated by 20-sim. 

### Inputs
Receives input from SRT part through XenoRos bridge.
Reads inputs through xeno_msg. See also the XenoRos bridge message.

### Outputs
Writes PWM values to FPGA. Logs variables through the Xeno logger, and
saves them as .bin files. 

### Build
Simply build through colcon using
`colcon build --packages-select assignment3_3`

### Run
In a terminal run the following command (BEFORE RUNNING ANYTHING ELSE!):
`./build/assignment3_2/XRF_test`

### Parameters
Does not take parameters.

### Core components 
* `run()`:      Main logic for updating the PWM values in the FPGA. Updates motor PWM through 20-sim generated controller, and logs both encoders and wheel positions.
* `unwrap()`:   Converts encoder data into pure angle measurement.
