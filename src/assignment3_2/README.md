Package assignment3_2
-----------------------------------------------
### Description 
This package implements the Firm real time part of assignment 3.2.
It contains the Xenomai 20-sim node using the FSM system. Requires the 
XRF2 bridge node and the relbot_adapter node to function properly. Also
contains the controller generated by 20-sim. 

### Inputs
Receives input from SRT part through XenoRos bridge.
Reads inputs through xeno_msg. See also the XenoRos bridge message.

### Outputs
Writes PWM values to FPGA. Logs variables through the Xeno logger, and
saves them as .bin files.

### Build
Simply build through colcon using
`colcon build --packages-select assignment3_2`

There is also a shell script provided that builds the package and all dependencies.
This is done through setting the shell to an executable, and then running it.
The shell script must be placed in the base workspace, and must NOT be given sudo access.
Run the following from the ros2_ws:
`chmod +X build_pi_3_2.sh`
`./build_pi_3_2.sh`
You may need to set permisson for the file, so use
`chmod 777 build_pi_3_2.sh`, or something similar.

Note that you still have to source `install/setup.bash` after this shell script.

### Run
In a terminal run the following command (BEFORE RUNNING ANYTHING ELSE!):
`./build/assignment3_2/assignment3_2`

### Parameters
Does not take parameters.

### Core components 
* `run()`: Main logic for updating the PWM values in the FPGA. Also saves logger data.
